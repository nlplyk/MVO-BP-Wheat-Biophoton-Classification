# MVO-BP Neural Network for Wheat Variety Classification Using Biophoton Delayed Luminescence Signals

This repository provides the official implementation, dataset, and results for the paper:
**Non-destructive Wheat Variety Classification Using Biophoton Signals Based on MVO-Optimized BP Neural Network**

## Overview
- **Task**: Non-destructive classification of 6 wheat varieties
- **Signal**: Delayed luminescence biophoton time-series data
- **Model**: MVO (Multi-Verse Optimizer) optimized BP neural network
- **Accuracy**: 98.67%
